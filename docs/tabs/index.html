This means you simply have to use the data-jd-tabs attibute — along with some basic accessibility attributes — to create some basic tabs!

A bunch of CSS

As this is a fairly complicated component, there is wuite a bit of CSS, but most of it is up to you

Parameters

A quick summary of the attributes used on the tabs container itself:

Attribute Role Default value
data-jd-tabs Creates the tabs No value
data-delay Defines the delay (in ms) to focus a tab when using arrows to navigate 300

Now for the attributes used on the tabs buttons:

Attribute Role Default value
role="tab" Required to make the tab system work
aria-controls The ID of the tab panel the tab controls Up to you
data-deletable Makes the whole tab deletable by using the DEL key false

And finally the attributes used on the panels:

Attribute Role Default value
role="tabpanel" Required to make the tab system work
id The ID of the tab panel (must be the same as the aria-controls attribute used in the button Up to you
